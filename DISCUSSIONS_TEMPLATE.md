## ğŸš€ Quick Start

New to Rune? Here's how to get started:

```bash
# Install Rune
curl -fsSL https://raw.githubusercontent.com/ferg-cod3s/rune/main/install.sh | bash

# Or with Homebrew
brew install --cask ferg-cod3s/tap/rune

# Interactive setup
rune init --guided

# Start your first session
rune start
```
